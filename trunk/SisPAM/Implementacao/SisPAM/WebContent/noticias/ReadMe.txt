-------------------------------------------------------------
	Como Utilizar
-------------------------------------------------------------

    Arquivos

      - index.htm
      - scripts/functions.js
      - scripts/stylesheet.css


	Este é um sistema simples de Notícias em RollOver
    (rolante). É muito simples sua instalação no seu text/script
    html. 

    --- 1° Passo ---

	Copiar a pasta 'Scripts' para a raiz do seu site, ou seja,
        para o local onde a sua página está guardada.
	Inserir na tag '<head> <!-- Insira aqui //--> </head>' da página
        onde funcionará o sistema as seguintes linhas de códigos:

	    - <script src="scripts/functions.js" type="text/javascript"></script>
	    - <link rel="stylesheet" href="scripts/stylesheet.css">

	        Estamos inserindo os arquivos que serão utilizados pelo sistema.
	        Confira se nenhum nome (funções/js e classes/css) coincidem
	     com alguma já inserida por você.

    --- 2° Passo ---

	Crie um objeto '<form name="form">' logo abixo da tag '<body>'
        para que abranja todo o documento.

	    <body>
	    <form name="form">

	Veja o código fonte da página 'index.htm' e copie o que está
        entre '<!-- Início //-->' e '<!-- Fim //-->' inserindo logo após no
        text/script html de sua página.

    --- 3° Passo ---

	Para funcionar corretamente, precisará que os arquivos estejam
        inclusos, o objeto '<form name="form">' esteja criado e preferencialmente
        todas as tags fechadas.


-------------------------------------------------------------
	Importante
-------------------------------------------------------------

        - Todas as cores editadas estão escritas no arquivo 'stylesheet.css'
        - As funções escritas no arquivo 'functions.js' não podem ser alteradas
        - O nome do 'objeto checkbox (cbScroll)' não pode ser mudado
        - Tudo que compõe a tag '<marquee>' neste texto não pode ser mudado

        - Este sistema não se limita a imprimir somente textos, pode, também,
          ser colocado imagens, mas com tamanho pré-estabelecido, cuja
          dimensão seria da largura de, no máximo, 125px (width="125")


-------------------------------------------------------------
	Você pode fazer mais
-------------------------------------------------------------

	Se você sabe um pouco de ASP ou PHP, verá que é fácil
        adaptar esse script ao que você precisa. Percebemos que cada
        bloco de mensagem compõe-se de uma tabela independente,
        facilitando todo o trabalho. Assim, na hora de usar o repetidor
        (Loop) que imprimirá as mensagens, o layout do sistema não será
        alterado
